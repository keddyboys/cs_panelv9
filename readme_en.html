Counter-Strike Server Status v2.3 Infusion for PHP-Fusion v9.00
What it does:

With this infusion you're able to check status an Counter-Strike server.
Installation:
  1. Upload folder files in the root folder.

  2. Go to Administration > System admin, click the "infusions" icon.

  3. Find in list Counter-Strike and click Infuse.

  4. Go to Administration > Infusions, Click Counter-Strike Admin and enter Settings.
Uninstallation:
  1. Go to Administration > System admin, click the "infusions" icon.

  2. Find in list Counter-Strike and click defuse.

  3. Delete folder counter_strike_panel from your infusions folder.
