Status server Counter-Strike v2.3 Infuzie pentru PHP-Fusion v9.00
Ce face:

Cu această infuzie vei putea verifica statusul unui server de Counter-Strike.
Un panou şi o legătură sait vor fi adăugate.
Instalare:
  1. Încarcă directorul files în directorul părinte al saitului tău.

  2. Du-te la Panoul de administrare > Administrare sistem, apasă pe iconiţa Infuzii.

  3. Găseşte Counter-Strike şi apasă infuzează.

  4. Du-te la Panoul de administrare > Infuzii apasă Administrare Counter-Strike şi introdu setările.
Dezinstalare:
  1. Du-te la Panoul de administrare > Administrare sistem, apasă pe iconiţa Infuzii.

  2. Găseşte Counter-Strike şi apasă defuzează.

  3. Şterge directorul counter_strike_panel din directorul infusions al saitului tău.
